# Software introduction

Computerhousekeeper is a software for managing computers

## Software development environment:

1. Windows10
2. Python3.9.0
3. PyQt5
4. PyCharm

# Software features

1. Get process information
2. Visualize CPU and memory usage
3. Get the boot entry
4. Scan for viruses
5. Search for files
6. Remove junk files
7. Download files
8. Manage Software

# Software reference

1. Use psutil to get process information
2. Using pyqt5 to design interface
3. Using pyqtgraph for data visualization
4. Source of virus library https://www.virscan.org/reportlist

# Software usage

Start `computerhousekeeper1.0.py`
